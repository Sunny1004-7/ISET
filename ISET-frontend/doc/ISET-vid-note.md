## 稿子

大家好！今天，我将代表我们的小组，向大家展示我们的项目——智慧教育系统。我们的项目旨在通过知识图谱、图数据库、大模型和多 Agent 等技术，让教育教学变得更加智慧。
首先，让我带大家从登录界面开始。登录后，我们来到了课程主页。课程主页展示了各类课程信息，包括课程名称、教师信息以及课程大纲。
特别值得一提的是，课程大纲中的知识图谱是我们项目的核心成果之一。我们通过自动构建和维护知识图谱，将教师上传的课件、视频、文本资料等融入其中，使课程内容更加丰富和结构化。
接下来，我们进入班级主页。班级主页展示了学生名单、班级通知以及班级动态。通过班级主页，教师可以方便地管理班级事务，学生也可以及时了解班级信息和动态。
此外，我们还为大家展示了智能问答页面。这一功能是我们项目的另一大亮点。通过对接学生行为数据库和知识图谱数据库 Agent，我们融合大模型的智能问答能够结合课程内容和学生学习行为，为教师备课提供便利，同时给予学生学习方向和重点的指导和建议。
在演示视频中，大家可以看到智能问答的具体操作过程。无论是教师还是学生，都可以通过智能问答获取所需的信息，实现教育教学的个性化定制。通过与学生学习行为数据库和知识图谱数据库的深度整合，我们的智能问答系统能够为教师提供备课支持，同时为学生提供个性化的学习指导和建议。它不仅提高了教学的针对性，也提升了学习的效率。
我们的智慧教育系统致力于让教育教学变得更加智慧、高效和个性化。通过前后端技术、知识图谱、图数据库、大模型和多 Agent 等技术的融合，我们为教育事业的未来发展贡献了一份力量。
最后，感谢大家的关注和支持！我们相信，在不久的将来，大语言模型将成为教育教学的重要工具，为每一个学生提供更加优质、个性化的教育。而我们的智慧教育系统，正是对这一未来的探索和实践。
谢谢大家！
